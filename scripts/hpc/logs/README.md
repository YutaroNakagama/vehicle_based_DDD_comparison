# HPC Logs Directory

This directory contains log files generated by HPC jobs. These files are ignored by git.

## Directory Structure

```
logs/
├── train/           # Training job logs
│   ├── Lstm/        # LSTM model training logs
│   ├── RF/          # Random Forest training logs
│   ├── SvmA/        # SVM (ANOVA kernel) training logs
│   └── SvmW/        # SVM (Wavelet features) training logs
├── preprocess/      # Preprocessing job logs
├── evaluate/        # Evaluation job logs
├── imbalance/       # Imbalance handling experiment logs
│   ├── BalancedRF/
│   ├── EasyEnsemble/
│   └── SMOTE/
└── monitor/         # Job monitoring and automation logs
```

## Notes

- All files in this directory are ignored by git (see `.gitignore`)
- Log files are named with job IDs for traceability
- Old logs can be safely deleted to free up disk space
