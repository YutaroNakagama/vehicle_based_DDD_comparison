config module

Configuration settings for the DDD pipeline.

This module centralizes configuration parameters for driver drowsiness detection, including dataset paths, model options, sampling rates, wavelet filters, and hyperparameter tuning.

Notes

All variables are defined as module-level constants. They are imported directly by other modules in the pipeline (e.g., preprocessing, training, evaluation).

config.DATASET_PATH = '../../dataset/mdapbe/physio'

Base path to the raw physiological data (e.g., SIMlsl, EEG).

Type:

str

config.DATA_PROCESS_CHOICES = ['SvmA', 'SvmW', 'Lstm', 'common']

Supported data processing strategies/models for preprocessing.

Type:

list of str

config.INTRIM_CSV_PATH = './data/interim'

Directory to store interim processed CSV files.

Type:

str

config.KSS_BIN_LABELS = [1, 2, 3, 4, 5, 7, 8, 9]

Original Karolinska Sleepiness Scale (KSS) labels in dataset.

Type:

list of int

config.KSS_LABEL_MAP = {1: 0, 2: 0, 3: 0, 4: 0, 5: 1, 6: 1, 7: 1, 8: 1, 9: 1}

Mapping of KSS labels to binary states (0=Alert, 1=Drowsy).

Type:

dict

config.MODEL_CHOICES = ['SvmA', 'SvmW', 'Lstm', 'RF', 'BalancedRF', 'DecisionTree', 'AdaBoost', 'GradientBoosting', 'XGBoost', 'LightGBM', 'CatBoost', 'LogisticRegression', 'SVM', 'K-Nearest Neighbors', 'MLP']

Supported machine learning models for training and evaluation.

list of str

config.MODEL_PKL_PATH = './model/'

str

config.MODEL_WINDOW_CONFIG = {'Lstm': {'step_sec': 1.5, 'window_sec': 3}, 'SvmA': {'step_sec': 1.5, 'window_sec': 3}, 'SvmW': {'step_sec': 1.5, 'window_sec': 3}, 'common': {'step_sec': 1.5, 'window_sec': 3}}

Windowing configuration (window length and step size in seconds).

Type:

dict

config.N_TRIALS = 100

Number of trials for hyperparameter optimization using Optuna.

Type:

int

config.OUTPUT_SVG_PATH = './output/svg'

Type:

str

config.PROCESS_CSV_PATH = './data/processed'

Directory to store final processed CSV files ready for model training.

Type:

str

int

Sampling rate for SIMLSL (Simulated Lane-keeping System) data [Hz].
